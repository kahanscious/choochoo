# Choochoo for Codex

## Install

```bash
git clone https://github.com/kahanscious/choochoo ~/.codex/choochoo
mkdir -p ~/.agents/skills
ln -s ~/.codex/choochoo/skills ~/.agents/skills/choochoo
```

Restart Codex.

## Update

```bash
cd ~/.codex/choochoo && git pull
```

## Uninstall

```bash
rm ~/.agents/skills/choochoo
```
